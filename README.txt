This folder contains a 360 interactive web application generated by CloudPano, available at http://www.cloudpano.com.

Usage
-----

The main application file is `index.html`. Many browsers have issues
when using `file://` URLs to load images. Therefore, if you were to open index.html in your browser on your local computer, 
it will not load due to browser security policies. 

To properly view the application it is necessary to use a web server.

Deployment
----------

To deploy the application, upload the contents of the folder into
a hosting service.
